# movie-recommender
A content-based movie recommendation system using Python and cosine similarity.

## Dataset
Based on the IMDB 5000 Movie Dataset (CSV: 'movie-metadata 2.csv')

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity

## How to Run
Install dependencies:

```bash
pip install pandas scikit-learn
